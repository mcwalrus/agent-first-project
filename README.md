# Agent-First Software Tutorial

Learn to run coding agents in **loops** and **harnesses**. You'll learn how you can provide agents with **automated feedback** so that when agents fail they can try again until they pass. You don’t need to wait for a newer or smarter model for that - you need a project that tells the agent when something’s off. This tutorial provides an example setup with agent tooling so you can **prove what AI is capable on a real project yourself**.
